Smart Attendance System Using Raspberry Pi

The Smart Attendance System Using Raspberry Pi involved designing an attendance system on Raspberry Pi computer by implementing RFID technology to authorize and record attendance, creating a database with unique IDs using MySQL, configuring the RFID reader to read the unique IDs stored in each RFID card, and programming the Raspberry Pi module to accurately count attendance, store data, and finally to generate an attendance report.

Smart Gas Cylinder Monitoring and Management System

The Smart Gas Cylinder Monitoring and Management System involved developing an embedded system based on the Arduino platform to monitor gas cylinder and manage gas leakage, integrating Load Cell Weight Sensor, MQ5 Gas Sensor, and Wi-Fi module with Arduino, and programming the Arduino microcontroller to send a signal to the solenoid valve to automatically close the cylinder valve upon detecting gas leakage and sending an alert in case of any gas leakage and automatic closure of cylinder valves.
