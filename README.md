# Portfolio Website

## Project Overview
This is my Flask-based portfolio website repository, showcasing my projects and skills. It serves as a personal space to present and update my works, share ideas, and connect with others.

## Features
- Responsive design
- Contact form
- Project showcase with images and descriptions
- Blog section for sharing thoughts and articles

## Tech Stack
- **Front-end:** HTML5, CSS3, JavaScript, Bootstrap
- **Back-end:** Flask, Python
- **Database:** SQLite
- **Hosting:** Any cloud provider

## Installation Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/Kirankumar1597/portfolio.git
   cd portfolio
   ```
2. Set up a virtual environment:
   ```bash
   python -m venv venv
   source venv/bin/activate  # On Windows use `venv\Scripts\activate`
   ```
3. Install the required packages:
   ```bash
   pip install -r requirements.txt
   ```

## Usage
To run the application locally, use the following command:
```bash
flask run
```
